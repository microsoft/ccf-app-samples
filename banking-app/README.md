# banking-app

## How to run the demo

```bash
$ cd banking-app
$ npm i # Necessary only for the first time
$ ./prepare_demo.sh

# In another terminal
$ cd banking-app
$ ./demo.sh
$ ./show_app_log.sh # You can see the application log
```

## How to run test
```bash
$ cd banking-app
$ npm i # Necessary only for the first time
$ ./test.sh
```
